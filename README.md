# TOPFY - Music Recommendation System

Sistema completo de recomendação de músicas desenvolvido com Node.js, Express e arquitetura MVC.

## Características

- Arquitetura MVC completa
- Classes orientadas a objetos com encapsulamento
- Integração com MusicAPI externa
- Sistema de recomendação personalizado
- Interface moderna inspirada no Spotify
- Gerenciamento de preferências musicais
- Histórico de reprodução
- Biblioteca musical completa

## Instalação

\`\`\`bash
npm install
\`\`\`

## Executar

\`\`\`bash
npm start
\`\`\`

O servidor estará disponível em `http://localhost:3000`

## Desenvolvimento

\`\`\`bash
npm run dev
\`\`\`

## Estrutura do Projeto

\`\`\`
/models          - Classes do sistema (Music, User, MusicLibrary, etc)
/controllers     - Lógica de negócio e manipulação de requisições
/routes          - Rotas da aplicação
/views           - Templates EJS
/public          - Arquivos estáticos (CSS, JS)
\`\`\`

## Funcionalidades

- Cadastro e login de usuários
- Exploração de músicas
- Sistema de recomendação baseado em preferências
- Reprodução de músicas com tracking de popularidade
- Adição de músicas via API ou manualmente
- Gerenciamento de preferências de gêneros
- Histórico de reprodução
- Dashboard personalizado

## Tecnologias

- Node.js
- Express.js
- EJS
- Arquitetura MVC
- POO com encapsulamento
- MusicAPI Integration
